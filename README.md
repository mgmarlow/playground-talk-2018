# Angular Playground Talk 2018
Evolve your frontend workflows.

## Setup

Code:
```
cd ./code/
npm i
npm run playground
```
Navigate to `localhost:4201/`

Slides (created with [reveal.js](https://revealjs.com/#/)):
```
cd ./slides/
npm i
npm start
```
